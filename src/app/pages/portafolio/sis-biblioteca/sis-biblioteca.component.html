Contexto

Se desarrolla un sistema que permite la visualización de los libros de una biblioteca.


Propuesta de Solución

Se presenta esta propuesta que permite a los usuarios visualizar los libros, autores y precios que tiene la biblioteca.
El sistema restringe el acceso solo a usuarios registrados.


Objetivos del sistema

  1. Facilitar el acceso a la información
  2. Restringir el acceso de los usuarios
  3. Maximizar el rendimiento del sistema
  4. Diseño escalable
  5. Alta seguridad
