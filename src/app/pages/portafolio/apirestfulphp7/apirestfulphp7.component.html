Contexto

Se desarrolla un API RESTful que permite el envío y recepción de datos desde una BD MySQL.
Uso de Postman para las pruebas de endpoints.


Propuesta de Solución

Se presenta esta propuesta que permite a los usuarios visualizar los registros desde la BD.
El sistema restringe el acceso solo a usuarios registrados.


Objetivos del sistema

  1. Facilitar el acceso a la información
  2. Restringir el acceso de los usuarios
  3. Maximizar el rendimiento del sistema
  4. Diseño escalable
  5. Alta seguridad
